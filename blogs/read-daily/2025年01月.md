+ 当最后一个比特币被挖出时，会发生什么？这可以说是加密货币领域中最重要的问题。因为如果国家开始将比特币作为储备，它们的存在可能取决于这个问题的答案。
+ 在2009年比特币刚推出时，每个区块包含50个比特币。而今天，每个区块仅包含3.125个比特币。
+ 比特币减半大约每四年发生一次，并将持续到2140年，那时比特币的最大供应量——2100万枚，将会达到
+ 比特币最大供应量是2100万枚的原因被认为是因为2009年比特币推出时，全球货币总量的价值约为21万亿美元。如果这真是实际原因，那么这隐含的假设是每个比特币最终会价值100万美元。
+ 比特币的需求并没有保持不变；它一直在稳定增长，而新发行比特币的供应量则在稳定减少。结果是，比特币的价格每四年的涨幅远远超过两倍。
+ 实际上，自2009年以来，比特币价格已经增长超过一百万倍。
+ 比特币使得数字化存储和转移价值成为可能，并相较于其他数字货币（包括其他加密货币），提供了最高级别的所有权保障。
+ 区块奖励不仅仅来自新铸造的比特币。它还来自比特币交易费用。
+ 从技术上讲，比特币并没有交易费用，但用户可以选择向矿工支付小费，以激励他们将你的交易包含在他们即将创建的下一个区块中
+ 比特币将继续运行下去，因为交易费用将足以激励矿工继续添加带有交易的新区块。
+ 比特币区块链目前的大小已超过400GB。
+ 这就形成了一个悖论：比特币需求的增加导致费用上升，而费用上升又使区块链更难维护，并随着时间推移降低去中心化程度。
+ 到2032年，99%的比特币将被挖出
+ 比特币挖矿的成本会根据挖矿活动的规模而变化。如果比特币的挖矿成本长期高于比特币价格，那么挖矿活动将会减少。
+ 比特币区块链在技术上由两个实体驱动：处理交易的比特币矿工，以及添加和验证交易的比特币节点。并非所有节点都是比特币矿工
+ 运行比特币节点非常简单，数量呈指数级增长的个人和机构受到激励去运行诚实节点，以保护区块链
+ 许多比特币矿工的运营成本已经接近于零。
+ 大多数国家之间和国内的 BTC 交易可能都与规避制裁有关。
+ 有可能比特币在个人和机构层面的采用变得如此普及，以至于人们要求政府挖掘比特币以保持其功能性。
+ 当你意识到货币正在快速贬值时，你会发现比特币矿工理论上可以通过借法币来挖 BTC 永续运营
+ BTC真正的风险是比特币未能被个人和机构足够接受，从而无法引入新的激励机制
+ 2018年比特币减半，BTC 的价格需要保持在 14 万到 18 万美元之间，大多数矿工才能盈利
+ 未来几年能源价格飙升的风险，以及国家未能足够快地接受比特币的风险，客观上是比特币面临的更大威胁。
+ 黄金价格近年来大幅上涨的原因，恰恰是因为各国政府纷纷转向去中心化货币。
+ 黄金的市值约为 18 万亿美元。有趣的是，比特币若达到 18 万亿美元的市值，其价格将约为 100 万美元。
