+ 默认情况下，Rust 会从标准库中定义一组项目，并将其引入到每个程序的作用域中，这组项目被称为 prelude（前奏），它被尽可能保持小规模，专注于那些几乎在每个 Rust 程序中都会用到的功能，尤其是 trait
+ `&` 表示这个参数是一个引用，它提供了一种方法，可以让代码的多个部分访问同一块数据，而无需将数据多次复制到内存中
+ Cargo 对外部 crate 的协调能力正是它的强项所在， crate 类似node 的 module
+ 在下载完 crate 后，Rust 会先编译这些 crate，然后在依赖可用的情况下编译整个项目
+ 由于 `Cargo.lock` 文件对于可复现的构建非常重要，它通常会与项目中的其他代码一起提交到源码管理系统
+ 模式和 `match` 构造是 Rust 的强大功能：它们允许你表达代码可能遇到的各种情况，并确保你处理了所有这些情况。
